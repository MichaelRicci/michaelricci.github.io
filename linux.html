How to Install Kali Linux on VirtualBox

This guide provides a detailed step-by-step process for setting up Kali Linux using Oracle VirtualBox on your existing operating system. This is ideal for running Kali safely without altering your computer's actual operating system.

Requirements

  • Oracle VirtualBox: Download and install from https://www.virtualbox.org/.
  • Kali Linux ISO file: Download from https://www.kali.org/get-kali/.
  • Minimum of 4GB of RAM (8GB recommended) for the virtual machine.
  • 20-50 GB of free disk space on your host machine for the virtual disk.

Installation Steps

Step 1: Install VirtualBox

  • Download and install VirtualBox on your host machine following the instructions on the VirtualBox website.

Step 2: Set Up a New Virtual Machine

  • Open VirtualBox and create a new virtual machine named "Kali Linux" with type set to Linux and version set to Debian (64-bit).
  • Allocate at least 2048 MB of RAM.
  • Create a virtual hard disk using VDI (VirtualBox Disk Image), set to dynamically allocated with at least 20 GB.

Step 3: Configure the Virtual Machine

  • Under the system settings, set the boot order to start with the optical drive.
  • Increase the processor count to 2 for better performance if possible.
  • Mount the Kali Linux ISO file to the virtual optical drive under storage settings.
  • Set the network settings to NAT for internet access or as required for your setup.

Step 4: Install Kali Linux

  • Start the VM and select "Graphical Install" from the Kali menu.
  • Follow through the installation steps, setting up language, location, keyboard, network, user, disk partitions, and installing the GRUB bootloader.

Step 5: Finalize Installation

  • Restart the VM, ensuring the ISO is unmounted, to boot into your new Kali Linux installation.

Step 6: Post Installation

  • Install VirtualBox Guest Additions for enhanced functionality and performance.
